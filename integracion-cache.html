Guía de integración: caché + loading progresivo

1. Incluir el archivo en tu HTML

<!-- Antes del cierre </body> -->
<script src="cotizador-cache.js"></script>

2. Reemplazar tu función de búsqueda actual

Antes tenías algo como:

// ANTES (sin caché)
async function buscar() {
  const res = await fetch('/api/cotizar', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ origen, destino, fechaIda, adultos })
  });
  const json = await res.json();
  mostrarResultados(json.vuelos);
}

Reemplázala por:

// DESPUÉS (con caché + loader progresivo)
const API_URL = 'https://tu-app.onrender.com'; // ← tu URL real

async function buscar() {
  const origen   = document.getElementById('origen').value;
  const destino  = document.getElementById('destino').value;
  const fecha    = document.getElementById('fecha').value;
  const adultos  = parseInt(document.getElementById('adultos').value) || 1;

  const fromCache = CostamarCache.isCached(origen, destino, fecha, adultos);

  try {
    const vuelos = await buscarVuelosConCache(origen, destino, fecha, adultos, API_URL);

    mostrarResultados(vuelos);

    if (fromCache) {
      mostrarBadgeCache('resultados-container'); // badge verde "desde caché"
    }

  } catch (err) {
    alert('Error: ' + err.message);
  }
}

// Conectar al botón
document.getElementById('btn-buscar').addEventListener('click', buscar);

3. HTML mínimo requerido

<!-- Inputs (ya los tenés) -->
<input id="origen"   type="text" />
<input id="destino"  type="text" />
<input id="fecha"    type="date" />
<input id="adultos"  type="number" value="1" />
<button id="btn-buscar">Buscar vuelos</button>

<!-- Contenedor de resultados (el loader busca este ID para el badge) -->
<div id="resultados-container"></div>
El loader se crea automáticamente. No hace falta agregar HTML para el overlay — CostamarLoader.init() lo inserta en el <body> la primera vez que se llama.

4. Personalización opcional

// Cambiar TTL del caché (default: 5 min)
CostamarCache.TTL = 10 * 60 * 1000; // 10 minutos

// Cambiar mensajes del loader
CostamarLoader.STEPS = [
  { msg: 'Consultando tarifas...', pct: 20 },
  { msg: 'Revisando disponibilidad...', pct: 50 },
  { msg: 'Preparando resultados...', pct: 80 },
];

// Limpiar caché manualmente (ej: botón "limpiar caché")
document.getElementById('btn-limpiar').addEventListener('click', () => {
  CostamarCache.purge();
  alert('Caché limpiado');
});

5. Compatibilidad con el caché del backend

El backend (Flask) también tiene caché de 5 minutos en memoria RAM. El caché del frontend vive en el navegador del usuario: si se reinicia Render, el backend pierde su caché pero el frontend no. Ambos trabajan en paralelo y se complementan.
